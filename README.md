# PGO Git Start

Simple project for learning Git, GitHub and branch workflow.

Author: Piotr Pękalski

Goal:
Practice creating a local repository, making commits, working on branches and merging them into main.





## Project structure

- README.md — project description
- src/Main.java — simple application entry point

## Commands used

- git init
- git status
- git add .gitignore README.md src/Main.java
- git commit -m "Initialize repository and add project structure"
- git remote add origin https://github.com/s35559-del/-pgo-git-start.git
- git push -u origin main
- git checkout -b feature/readme
